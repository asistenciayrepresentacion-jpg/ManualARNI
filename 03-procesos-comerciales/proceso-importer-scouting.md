# Proceso Operativo: Importer Scouting

## 1. Objetivo del proceso

Este proceso establece los pasos operativos para ejecutar un servicio de Importer Scouting de forma ordenada, trazable y validable.

Su propósito es transformar una necesidad comercial del cliente en una lista calificada de prospectos, contactos priorizados, mensajes de acercamiento y recomendaciones de siguiente paso.

---

## 2. Resultado esperado

Al finalizar el proceso, ARNI debe contar con:

1. Ficha clara del producto.
2. Mercado objetivo definido.
3. Perfil ideal de importador o distribuidor.
4. Base de prospectos identificados.
5. Clasificación A/B/C.
6. Contactos priorizados.
7. Mensajes de acercamiento.
8. Registro de seguimiento.
9. Reporte ejecutivo para el cliente.
10. Recomendación de siguiente fase.

---

## 3. Insumos necesarios

Antes de iniciar, el cliente debe entregar o confirmar:

* Nombre del producto.
* Descripción comercial.
* Categoría.
* Presentaciones.
* Capacidad de producción.
* Precio estimado o rango objetivo.
* Certificaciones disponibles.
* País o región objetivo.
* Material comercial.
* Página web o catálogo.
* Perfil de cliente ideal.
* Restricciones logísticas o regulatorias conocidas.
* Objetivo comercial del proyecto.

---

## 4. Etapas del proceso

## Etapa 1: Diagnóstico inicial del producto

### Objetivo

Confirmar si existe información suficiente para iniciar la búsqueda de importadores.

### Actividades

1. Revisar ficha técnica o descripción del producto.
2. Confirmar categoría comercial.
3. Identificar diferenciadores.
4. Revisar certificaciones disponibles.
5. Confirmar capacidad de producción.
6. Validar si existe precio de exportación o rango esperado.
7. Detectar vacíos de información.

### Entregable interno

Ficha inicial del producto.

### Validación humana requerida

ARNI debe confirmar que la información del producto es suficiente y no contiene promesas comerciales, regulatorias o aduanales no validadas.

---

## Etapa 2: Definición del mercado objetivo

### Objetivo

Delimitar el mercado donde se buscarán prospectos.

### Actividades

1. Definir país objetivo.
2. Definir región, ciudad o canal prioritario.
3. Identificar tipo de comprador buscado.
4. Confirmar si se busca importador, distribuidor, retailer, broker, food service u otro canal.
5. Evaluar si conviene iniciar con mercado regional antes de mercado nacional.

### Entregable interno

Definición del mercado objetivo y canal prioritario.

---

## Etapa 3: Construcción del perfil ideal de contraparte

### Objetivo

Definir qué tipo de empresa debe ser considerada prospecto relevante.

### Criterios sugeridos

* Maneja productos similares o complementarios.
* Atiende el canal objetivo.
* Tiene operación activa.
* Tiene cobertura geográfica compatible.
* Tiene experiencia con productos importados.
* Tiene señales de capacidad comercial.
* Tiene canales de contacto verificables.
* Puede tener interés en nuevos proveedores.

### Entregable interno

Perfil ideal de importador o distribuidor.

---

## Etapa 4: Búsqueda de prospectos

### Objetivo

Construir una base inicial de posibles contrapartes.

### Fuentes posibles

* Directorios comerciales.
* Ferias y exposiciones.
* Cámaras empresariales.
* Asociaciones sectoriales.
* Sitios web de distribuidores.
* Marketplaces B2B.
* Bases internas.
* Referencias de misiones comerciales.
* Búsquedas públicas estructuradas.

### Campos mínimos de registro

| Campo               | Descripción                                      |
| ------------------- | ------------------------------------------------ |
| Empresa             | Nombre del prospecto                             |
| País/ciudad         | Ubicación principal                              |
| Tipo de contraparte | Importador, distribuidor, retailer, broker, etc. |
| Sector              | Categoría principal                              |
| Sitio web           | Fuente pública                                   |
| Contacto            | Persona identificada                             |
| Cargo               | Puesto del contacto                              |
| Email               | Correo disponible                                |
| Teléfono            | Teléfono disponible                              |
| Fuente              | De dónde se obtuvo la información                |
| Observaciones       | Notas relevantes                                 |

---

## Etapa 5: Clasificación por fit comercial

### Objetivo

Determinar qué tan compatible es cada prospecto con el producto del cliente.

### Criterios de fit

| Criterio                | Pregunta clave                                                    |
| ----------------------- | ----------------------------------------------------------------- |
| Categoría               | ¿Maneja productos similares o complementarios?                    |
| Canal                   | ¿Atiende el canal que busca el cliente?                           |
| Cobertura               | ¿Opera en la región objetivo?                                     |
| Posicionamiento         | ¿Su portafolio es compatible con el precio y perfil del producto? |
| Experiencia importadora | ¿Muestra señales de trabajar con productos internacionales?       |
| Relevancia              | ¿Tiene sentido comercial contactarlo?                             |

### Resultado

Cada prospecto se clasifica como:

* A: Alta prioridad.
* B: Prioridad media.
* C: Baja prioridad.

---

## Etapa 6: Calificación de capacidad

### Objetivo

Separar prospectos visibles de prospectos realmente útiles.

### Señales de capacidad

* Portafolio activo.
* Canales de venta claros.
* Presencia comercial verificable.
* Equipo o contactos identificables.
* Cobertura regional o nacional.
* Relación con marcas similares.
* Participación en ferias o asociaciones.
* Información pública consistente.

### Validación humana requerida

No se debe considerar un prospecto como prioritario solo porque aparece en internet. Debe existir evidencia mínima de operación y compatibilidad.

---

## Etapa 7: Preparación del mensaje de contacto

### Objetivo

Crear un mensaje breve, claro y profesional para iniciar conversación.

### El mensaje debe incluir

1. Presentación breve.
2. Producto ofrecido.
3. Razón de relevancia para el prospecto.
4. Solicitud de conversación.
5. Disponibilidad para compartir más información.

### Restricciones

El mensaje no debe prometer:

* Exclusividad.
* Precios finales no validados.
* Cumplimiento regulatorio no confirmado.
* Capacidad de entrega no autorizada.
* Condiciones comerciales no aprobadas por el cliente.

---

## Etapa 8: Contacto y seguimiento

### Objetivo

Registrar todas las interacciones de forma ordenada.

### Estados sugeridos

| Estado                | Significado                                 |
| --------------------- | ------------------------------------------- |
| Identificado          | Prospecto encontrado                        |
| Validado              | Prospecto revisado y aceptado para contacto |
| Contactado            | Ya se envió mensaje                         |
| Respondió             | Hubo respuesta                              |
| Interesado            | Mostró interés inicial                      |
| Requiere información  | Solicitó datos adicionales                  |
| Reunión agendada      | Existe reunión programada                   |
| En evaluación         | Está revisando información                  |
| No interesado         | Rechazó o no aplica                         |
| Oportunidad potencial | Puede avanzar a siguiente fase              |

---

## Etapa 9: Conversión a oportunidad

### Objetivo

Definir cuándo un prospecto deja de ser contacto y se convierte en oportunidad comercial.

### Señales mínimas

Un prospecto puede convertirse en oportunidad si:

* Solicita ficha técnica.
* Solicita precios.
* Solicita muestras.
* Pregunta por volúmenes.
* Agenda reunión.
* Pregunta por logística.
* Pregunta por representación.
* Pide información comercial adicional.
* Comparte datos de compra o distribución.

### Regla operativa

No todo prospecto contactado es una oportunidad. Solo debe convertirse cuando exista una señal clara de interés.

---

## Etapa 10: Reporte al cliente

### Objetivo

Presentar resultados útiles para la toma de decisiones.

### Estructura sugerida del reporte

1. Resumen ejecutivo.
2. Objetivo del scouting.
3. Producto analizado.
4. Mercado objetivo.
5. Metodología.
6. Número de prospectos identificados.
7. Clasificación A/B/C.
8. Top prospectos recomendados.
9. Hallazgos relevantes.
10. Riesgos detectados.
11. Recomendaciones.
12. Siguiente paso sugerido.

---

## 5. Uso de IA en el proceso

La IA puede apoyar en:

* Ordenar información del producto.
* Sugerir criterios de búsqueda.
* Clasificar prospectos.
* Redactar mensajes iniciales.
* Resumir información pública.
* Preparar reportes.
* Identificar riesgos.
* Convertir notas en acciones de seguimiento.

---

## 6. Controles humanos obligatorios

Una persona debe validar:

1. Que el prospecto exista.
2. Que la empresa sea compatible.
3. Que el contacto sea correcto.
4. Que la información pública sea actual.
5. Que el mensaje sea adecuado.
6. Que no se prometan resultados no autorizados.
7. Que la oportunidad tenga señales reales antes de avanzar.

---

## 7. Indicadores del proceso

| Indicador                 | Descripción                                       |
| ------------------------- | ------------------------------------------------- |
| Prospectos identificados  | Número total de empresas encontradas              |
| Prospectos validados      | Empresas que sí cumplen criterios mínimos         |
| Prospectos A              | Empresas de alta prioridad                        |
| Prospectos contactados    | Empresas a las que se envió mensaje               |
| Respuestas recibidas      | Prospectos que respondieron                       |
| Reuniones generadas       | Conversaciones agendadas                          |
| Oportunidades potenciales | Prospectos con interés real                       |
| Tasa de conversión        | Respuestas o reuniones sobre contactos realizados |

---

## 8. Riesgos frecuentes

* Hacer búsqueda sin producto bien definido.
* Confundir empresas grandes con buenos prospectos.
* No validar el canal.
* No confirmar capacidad del cliente.
* Contactar sin mensaje profesional.
* No registrar seguimiento.
* Prometer cierre comercial.
* Depender de IA sin revisión humana.

## 10. Regla final

El objetivo del Importer Scouting no es entregar una lista larga de contactos.

El objetivo es entregar una base calificada, priorizada y accionable que ayude al cliente a decidir con quién hablar, qué validar y qué siguiente paso comercial tomar.
