# Criterios de Calificación de Oportunidades

Este documento define los criterios base para evaluar oportunidades comerciales de ARNI Consulting.

## Objetivo

Establecer una metodología simple para clasificar prospectos, reuniones y oportunidades como Prioridad A, B o C.

## 1. Criterios principales

Toda oportunidad debe evaluarse con los siguientes criterios:

| Criterio | Pregunta clave |
|---|---|
| Necesidad clara | ¿La contraparte tiene un problema o interés concreto? |
| Decisor identificado | ¿Sabemos quién puede aprobar o influir en la decisión? |
| Capacidad probable de pago | ¿La empresa parece tener recursos para contratar? |
| Urgencia | ¿Existe una razón para actuar pronto? |
| Fit con ARNI | ¿ARNI puede aportar valor real? |
| Alcance posible | ¿Se puede definir un servicio claro? |
| Información disponible | ¿Tenemos suficiente información para avanzar? |
| Riesgo manejable | ¿Los riesgos comerciales u operativos son aceptables? |
| Siguiente paso | ¿Existe una acción concreta después del contacto? |

## 2. Sistema de puntuación

Cada criterio puede evaluarse con una puntuación de 0 a 2:

- 0 = No cumple.
- 1 = Cumple parcialmente.
- 2 = Cumple claramente.

## 3. Interpretación del puntaje

| Puntaje total | Clasificación | Acción recomendada |
|---|---|---|
| 15 a 18 | Prioridad A | Avanzar con seguimiento inmediato |
| 10 a 14 | Prioridad B | Solicitar más información o madurar oportunidad |
| 0 a 9 | Prioridad C | Mantener en observación o descartar temporalmente |

## 4. Prioridad A

Una oportunidad A tiene:

- Necesidad clara.
- Servicio ARNI aplicable.
- Contacto relevante.
- Posibilidad razonable de pago.
- Siguiente paso concreto.
- Riesgo manejable.

Acción recomendada:

- Agendar reunión.
- Preparar diagnóstico.
- Crear oportunidad formal.
- Preparar propuesta si el alcance está claro.

## 5. Prioridad B

Una oportunidad B tiene potencial, pero falta información.

Puede faltar:

- Decisor.
- Presupuesto.
- Urgencia.
- Alcance.
- Información del producto o empresa.
- Claridad sobre el servicio requerido.

Acción recomendada:

- Solicitar información adicional.
- Mantener seguimiento.
- Hacer reunión exploratoria.
- No preparar propuesta completa todavía.

## 6. Prioridad C

Una oportunidad C no tiene suficiente claridad o valor inmediato.

Puede tener:

- Bajo fit con ARNI.
- Falta de presupuesto.
- Interés muy exploratorio.
- Contacto sin poder de decisión.
- Expectativas poco realistas.
- Riesgo operativo alto.

Acción recomendada:

- Registrar.
- Mantener contacto futuro si conviene.
- No invertir tiempo significativo.
- Descartar temporalmente si no hay siguiente paso.

## 7. Señales de alerta

Antes de avanzar, revisar si existe alguna señal de riesgo:

- El prospecto pide muchos entregables gratis.
- No hay claridad sobre quién decide.
- Quiere resultados garantizados.
- No comparte información básica.
- Busca solo contactos sin estrategia.
- No tiene capacidad de seguimiento.
- El alcance cambia constantemente.
- El presupuesto parece incompatible con el esfuerzo requerido.

## 8. Principio de uso

La calificación no debe reemplazar el criterio profesional. Debe ayudar a ARNI a priorizar tiempo, proteger márgenes y enfocar seguimiento en oportunidades con mayor valor.
