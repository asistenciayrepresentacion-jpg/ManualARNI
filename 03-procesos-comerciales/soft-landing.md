# Proceso Operativo Soft Landing

## 1. Objetivo del proceso

El proceso operativo de Soft Landing tiene como objetivo guiar la ejecución práctica del servicio, desde la recepción del cliente hasta la definición de una ruta de entrada al mercado objetivo.

Su finalidad es ordenar información, reducir incertidumbre, validar condiciones iniciales, identificar actores relevantes y transformar el interés de entrada a mercado en un plan accionable.

Este proceso no sustituye asesoría legal, fiscal, migratoria, regulatoria, aduanal o técnica especializada. Cuando se detecten temas que requieran revisión profesional, ARNI debe canalizar al cliente con especialistas externos.

---

## 2. Resultado esperado del proceso

Al finalizar el proceso, ARNI debe entregar al cliente una visión estructurada sobre:

1. Nivel de preparación para entrar al mercado.
2. Principales brechas comerciales u operativas.
3. Actores relevantes identificados.
4. Riesgos y barreras iniciales.
5. Ruta recomendada de entrada.
6. Próximas acciones sugeridas.
7. Posibilidad de avanzar a Business Matchmaking, Pilot Validation, Trade Mission Support u otro servicio ARNI.

---

## 3. Entradas necesarias

Antes de iniciar, ARNI debe recopilar información mínima del cliente.

### 3.1 Información de la empresa

* Nombre de la empresa.
* País de origen.
* Sector.
* Producto o servicio.
* Sitio web.
* Persona responsable.
* Cargo.
* Datos de contacto.
* Experiencia previa en mercados internacionales.
* Mercados donde ya opera.

### 3.2 Información sobre la oferta

* Descripción del producto o servicio.
* Propuesta de valor.
* Diferenciadores.
* Presentación comercial.
* Catálogo.
* Ficha técnica, si aplica.
* Certificaciones, si aplica.
* Capacidad de producción o prestación.
* Precio, rango de precio o modelo comercial.
* Casos de éxito o referencias.

### 3.3 Información sobre el objetivo de entrada

* Mercado objetivo.
* Motivo para entrar a ese mercado.
* Tipo de entrada buscada.
* Horizonte de tiempo.
* Presupuesto aproximado.
* Tipo de contraparte deseada.
* Nivel de compromiso del equipo directivo.
* Restricciones comerciales, financieras u operativas.

## 5. Etapas del proceso

---

# Etapa 1: Recepción y definición del alcance

## Objetivo

Entender qué busca el cliente y delimitar qué cubrirá el servicio de Soft Landing.

## Actividades

1. Realizar reunión inicial con el cliente.
2. Confirmar mercado objetivo.
3. Identificar motivación de entrada.
4. Definir si el cliente busca vender, distribuir, invertir, representar, instalarse o validar.
5. Confirmar expectativas.
6. Identificar restricciones de presupuesto, tiempo y capacidad.
7. Aclarar qué temas quedan fuera del alcance.
8. Definir entregables esperados.
9. Registrar acuerdos iniciales.

## Preguntas clave

* ¿Por qué quiere entrar a este mercado?
* ¿Qué decisión quiere tomar al final del proceso?
* ¿Qué información necesita para avanzar?
* ¿Qué tipo de apoyo espera de ARNI?
* ¿Qué riesgos le preocupan más?
* ¿Qué plazo tiene para tomar decisiones?
* ¿Ya tiene contactos o antecedentes en el mercado?

## Entregable interno

Ficha inicial del cliente y alcance del Soft Landing.

## Criterio de avance

La etapa puede avanzar cuando ARNI tenga claridad sobre el objetivo, mercado, tipo de entrada y entregables esperados.

---

# Etapa 2: Perfilamiento del cliente

## Objetivo

Construir un perfil comercial y operativo del cliente para evaluar su nivel de preparación.

## Actividades

1. Solicitar información de empresa, producto, servicio y mercado.
2. Revisar materiales disponibles.
3. Evaluar claridad de propuesta de valor.
4. Identificar experiencia internacional previa.
5. Confirmar capacidad de respuesta.
6. Revisar si existen precios, fichas, catálogos o presentaciones.
7. Detectar información incompleta.
8. Registrar brechas iniciales.

## Campos mínimos

| Campo               | Descripción                                                           |
| ------------------- | --------------------------------------------------------------------- |
| Empresa             | Nombre legal o comercial.                                             |
| Sector              | Industria principal.                                                  |
| Oferta              | Producto, servicio o solución.                                        |
| Mercado objetivo    | País, región o ciudad prioritaria.                                    |
| Objetivo de entrada | Venta, distribución, inversión, alianza, representación o validación. |
| Capacidad           | Producción, operación, equipo, cobertura o recursos disponibles.      |
| Materiales          | Catálogo, pitch, sitio web, ficha, presentación o casos.              |
| Diferenciador       | Razón por la cual la oferta podría ser atractiva.                     |
| Restricciones       | Tiempo, presupuesto, certificaciones, idioma, capacidad o regulación. |

## Entregable interno

Perfil comercial del cliente.

## Criterio de avance

La etapa puede avanzar cuando el cliente tenga información suficiente para evaluar preparación y construir una hipótesis de entrada.

---

# Etapa 3: Evaluación de preparación mínima

## Objetivo

Determinar si el cliente está listo para iniciar acercamientos o si primero debe cerrar brechas.

## Actividades

1. Evaluar materiales comerciales.
2. Revisar claridad del pitch.
3. Validar si existe una propuesta de valor entendible para el mercado.
4. Revisar si el cliente puede responder a solicitudes de información.
5. Identificar brechas de idioma, presentación, precio o documentación.
6. Clasificar el nivel de preparación.
7. Recomendar acciones correctivas.

## Criterios de evaluación

| Criterio               | Pregunta de control                                               |
| ---------------------- | ----------------------------------------------------------------- |
| Oferta clara           | ¿Se entiende qué vende o propone el cliente?                      |
| Mercado definido       | ¿Sabe a qué país, región o segmento quiere entrar?                |
| Material comercial     | ¿Tiene presentación, catálogo o ficha?                            |
| Pitch ejecutivo        | ¿Puede explicar su propuesta en menos de 3 minutos?               |
| Capacidad de respuesta | ¿Puede contestar rápido a una contraparte?                        |
| Información comercial  | ¿Tiene precios, rangos, condiciones o modelo de negocio?          |
| Evidencia              | ¿Tiene clientes, casos, certificaciones o referencias?            |
| Adaptación             | ¿El material está preparado para el idioma y cultura del mercado? |

## Clasificación

| Nivel | Descripción                                                | Acción recomendada                                  |
| ----- | ---------------------------------------------------------- | --------------------------------------------------- |
| Alto  | Puede iniciar acercamientos con ajustes menores.           | Avanzar a análisis de mercado y actores.            |
| Medio | Tiene base útil, pero requiere mejoras antes de contactar. | Corregir brechas críticas.                          |
| Bajo  | No está listo para acercamientos formales.                 | Preparar materiales y propuesta antes de continuar. |

## Entregable interno

Matriz de preparación mínima.

## Criterio de avance

La etapa puede avanzar si el cliente tiene nivel Alto o Medio. Si tiene nivel Bajo, se recomienda pausar el Soft Landing y trabajar primero en preparación comercial.

---

# Etapa 4: Hipótesis de entrada al mercado

## Objetivo

Definir una hipótesis inicial sobre cómo podría entrar el cliente al mercado objetivo.

## Actividades

1. Analizar el tipo de oferta.
2. Identificar posibles canales de entrada.
3. Definir segmento objetivo.
4. Identificar posibles compradores, aliados o instituciones.
5. Formular hipótesis de entrada.
6. Validar si la hipótesis es realista.
7. Documentar supuestos principales.

## Tipos de hipótesis

* Entrada mediante distribuidor.
* Entrada mediante importador.
* Entrada mediante cliente directo.
* Entrada mediante socio estratégico.
* Entrada mediante representante local.
* Entrada mediante piloto comercial.
* Entrada mediante misión comercial.
* Entrada mediante validación institucional.
* Entrada mediante instalación local futura.

## Formato sugerido

| Elemento                | Descripción                                                          |
| ----------------------- | -------------------------------------------------------------------- |
| Mercado objetivo        | País, región o ciudad.                                               |
| Segmento prioritario    | Tipo de cliente o sector.                                            |
| Canal probable          | Distribuidor, comprador directo, aliado, institución, representante. |
| Justificación           | Razón por la cual ese canal parece viable.                           |
| Riesgos                 | Barreras o dudas principales.                                        |
| Información por validar | Datos que deben confirmarse antes de avanzar.                        |

## Entregable interno

Hipótesis preliminar de entrada.

## Criterio de avance

La etapa puede avanzar cuando exista una hipótesis clara y comprobable.

---

# Etapa 5: Análisis ejecutivo del mercado

## Objetivo

Revisar condiciones iniciales del mercado para confirmar si la hipótesis de entrada tiene sentido.

## Actividades

1. Investigar contexto general del sector.
2. Revisar tendencias relevantes.
3. Identificar canales de comercialización.
4. Mapear competidores o alternativas.
5. Identificar posibles barreras de entrada.
6. Detectar requisitos que requieren revisión especializada.
7. Identificar regiones o segmentos prioritarios.
8. Preparar resumen ejecutivo.

## Información a revisar

* Tamaño o relevancia general del sector.
* Tendencias de consumo o demanda.
* Canales existentes.
* Competidores relevantes.
* Precios de referencia, si aplica.
* Requisitos comerciales mínimos.
* Actores institucionales.
* Eventos, ferias o espacios de networking.
* Riesgos iniciales.
* Barreras regulatorias que requieran especialista.

## Entregable interno

Resumen ejecutivo de mercado.

## Criterio de avance

La etapa puede avanzar cuando ARNI tenga señales suficientes para priorizar actores y siguientes validaciones.

---

# Etapa 6: Mapeo de actores relevantes

## Objetivo

Identificar empresas, instituciones, aliados o contactos que puedan aportar información, validación o acceso al mercado.

## Actividades

1. Definir tipos de actores a buscar.
2. Construir base preliminar.
3. Clasificar actores por tipo.
4. Revisar relevancia.
5. Identificar fuente de información.
6. Registrar contacto disponible, si existe.
7. Clasificar prioridad A/B/C.
8. Seleccionar actores para posible acercamiento.

## Tipos de actores

* Compradores potenciales.
* Distribuidores.
* Importadores.
* Brokers.
* Integradores.
* Representantes.
* Cámaras empresariales.
* Asociaciones sectoriales.
* Agencias de promoción.
* Consultores locales.
* Hubs empresariales.
* Proveedores de servicios.
* Aliados institucionales.
* Empresas comparables.
* Expertos sectoriales.

## Campos mínimos de la base

| Campo          | Descripción                                              |
| -------------- | -------------------------------------------------------- |
| Nombre         | Empresa, institución o persona.                          |
| Tipo de actor  | Comprador, aliado, distribuidor, cámara, consultor, etc. |
| País o ciudad  | Ubicación principal.                                     |
| Sector         | Sector relacionado.                                      |
| Relevancia     | Razón por la cual puede ser útil.                        |
| Contacto       | Persona, correo, LinkedIn o formulario, si existe.       |
| Fuente         | Sitio web, directorio, evento, referencia u otra fuente. |
| Prioridad      | A, B o C.                                                |
| Próxima acción | Investigar, contactar, validar, descartar o monitorear.  |

## Entregable interno

Mapa inicial de actores relevantes.

## Criterio de avance

La etapa puede avanzar cuando exista una base priorizada de actores y al menos un grupo de contactos con potencial de validación.

---

# Etapa 7: Priorización de actores y oportunidades

## Objetivo

Seleccionar los actores más relevantes para validación, acercamiento o seguimiento.

## Actividades

1. Revisar la base de actores.
2. Evaluar fit con el objetivo del cliente.
3. Evaluar capacidad del actor.
4. Evaluar relevancia estratégica.
5. Evaluar accesibilidad.
6. Detectar riesgos de bajo fit.
7. Asignar prioridad.
8. Definir secuencia de acercamiento.

## Criterios de priorización

| Criterio      | Pregunta clave                                          |
| ------------- | ------------------------------------------------------- |
| Fit sectorial | ¿Opera en el sector relevante?                          |
| Fit de canal  | ¿Corresponde al canal que busca el cliente?             |
| Relevancia    | ¿Puede aportar información o avance útil?               |
| Capacidad     | ¿Tiene operación, influencia o experiencia comprobable? |
| Accesibilidad | ¿Existe una vía razonable de contacto?                  |
| Potencial     | ¿Podría convertirse en oportunidad futura?              |
| Riesgo        | ¿Hay señales de baja utilidad o pérdida de tiempo?      |

## Clasificación

| Prioridad | Descripción                                | Acción                                       |
| --------- | ------------------------------------------ | -------------------------------------------- |
| A         | Alta relevancia y buen fit.                | Contactar o preparar acercamiento.           |
| B         | Relevancia media o información incompleta. | Investigar más o contactar en segunda ronda. |
| C         | Bajo fit o utilidad limitada.              | Mantener como referencia o descartar.        |

## Entregable interno

Matriz de priorización de actores.

## Criterio de avance

La etapa puede avanzar cuando ARNI tenga una lista corta de actores prioritarios.

---

# Etapa 8: Preparación de materiales de acercamiento

## Objetivo

Preparar los mensajes y documentos mínimos para contactar actores o validar información.

## Actividades

1. Ajustar pitch del cliente.
2. Preparar mensaje institucional de ARNI.
3. Preparar mensaje de acercamiento al actor.
4. Preparar preguntas de validación.
5. Preparar one-pager o ficha ejecutiva.
6. Traducir o adaptar material si aplica.
7. Validar tono y alcance con el cliente.
8. Registrar versión final.

## Materiales posibles

* Correo de presentación.
* Mensaje de LinkedIn.
* One-pager del cliente.
* Ficha de producto o servicio.
* Pitch ejecutivo.
* Guion de llamada.
* Preguntas de validación.
* Breve presentación institucional.
* Resumen de objetivo de reunión.

## Entregable interno

Paquete de acercamiento.

## Criterio de avance

La etapa puede avanzar cuando los mensajes y materiales estén listos para usarse sin improvisación.

---

# Etapa 9: Validación con actores del mercado

## Objetivo

Obtener señales iniciales del mercado mediante conversaciones, respuestas, entrevistas o reuniones exploratorias.

## Actividades

1. Contactar actores priorizados.
2. Registrar respuestas.
3. Coordinar reuniones, si aplica.
4. Preparar briefing previo.
5. Realizar reunión o conversación.
6. Documentar aprendizajes.
7. Identificar objeciones.
8. Confirmar próximos pasos.
9. Clasificar resultado.

## Tipos de validación

* Respuesta por correo.
* Llamada exploratoria.
* Reunión virtual.
* Reunión presencial.
* Introducción institucional.
* Comentario de experto.
* Validación de canal.
* Validación de comprador.
* Validación de socio local.

## Campos de registro

| Campo            | Descripción                                                                   |
| ---------------- | ----------------------------------------------------------------------------- |
| Actor contactado | Nombre de empresa, institución o persona.                                     |
| Tipo de actor    | Comprador, aliado, distribuidor, cámara, etc.                                 |
| Fecha            | Fecha de contacto o reunión.                                                  |
| Resultado        | Sin respuesta, interesado, no interesado, informativo, oportunidad potencial. |
| Aprendizaje      | Información útil obtenida.                                                    |
| Objeción         | Duda, barrera o rechazo detectado.                                            |
| Próxima acción   | Enviar información, agendar, investigar, descartar o dar seguimiento.         |
| Responsable      | Persona encargada del seguimiento.                                            |

## Entregable interno

Matriz de validación de mercado.

## Criterio de avance

La etapa puede avanzar cuando existan señales suficientes para formular recomendaciones o cuando se agote la primera ronda de validación.

---

# Etapa 10: Análisis de hallazgos

## Objetivo

Interpretar la información obtenida y convertirla en conclusiones útiles para el cliente.

## Actividades

1. Revisar respuestas y reuniones.
2. Agrupar aprendizajes por tema.
3. Identificar señales positivas.
4. Identificar objeciones recurrentes.
5. Detectar brechas del cliente.
6. Evaluar si la hipótesis de entrada se confirma, ajusta o descarta.
7. Preparar conclusiones.
8. Definir posibles rutas de acción.

## Categorías de análisis

* Demanda potencial.
* Fit de producto o servicio.
* Canal más viable.
* Barreras comerciales.
* Barreras operativas.
* Barreras regulatorias por revisar.
* Nivel de preparación del cliente.
* Relevancia de actores contactados.
* Potencial de siguientes reuniones.
* Necesidad de aliados externos.

## Entregable interno

Resumen de hallazgos.

## Criterio de avance

La etapa puede avanzar cuando ARNI tenga conclusiones claras y respaldadas por la información recopilada.

---

# Etapa 11: Recomendación de ruta de entrada

## Objetivo

Definir la ruta más conveniente para que el cliente avance después del Soft Landing.

## Actividades

1. Revisar objetivo inicial.
2. Comparar hipótesis contra hallazgos.
3. Identificar rutas posibles.
4. Evaluar ventajas, riesgos y requisitos.
5. Recomendar siguiente paso.
6. Definir acciones inmediatas.
7. Asignar responsables.
8. Preparar plan de seguimiento.

## Rutas posibles

| Ruta                       | Cuándo aplica                                                                        |
| -------------------------- | ------------------------------------------------------------------------------------ |
| Business Matchmaking       | Cuando hay actores con fit y posibilidad de reuniones B2B.                           |
| Pilot Validation           | Cuando se necesita probar producto, canal, precio, aceptación o rotación.            |
| Trade Mission Support      | Cuando conviene organizar visita, agenda o misión comercial.                         |
| Market Readiness           | Cuando el cliente necesita fortalecer materiales, precio, propuesta o documentación. |
| Importer Scouting          | Cuando el principal objetivo es encontrar importadores o distribuidores.             |
| Alianza local              | Cuando se requiere socio, representante, integrador o consultor especializado.       |
| Pausa estratégica          | Cuando las brechas son demasiado altas para avanzar.                                 |
| Replanteamiento de mercado | Cuando el mercado objetivo no parece prioritario o viable.                           |

## Entregable interno

Ruta recomendada de entrada.

## Criterio de avance

La etapa puede avanzar cuando exista una recomendación clara, realista y accionable.

3. Perfil del cliente.
4. Hipótesis inicial de entrada.
5. Evaluación de preparación.
6. Análisis ejecutivo del mercado.

10. Riesgos y barreras detectadas.
11. Brechas del cliente.

13. Próximas acciones.

## Entregables finales posibles

* Reporte ejecutivo.

* Mapa de actores.

* Matriz de validación.

* Minutas de reuniones.

* Plan de seguimiento.

## Criterio de cierre

El proceso se considera cerrado cuando el cliente recibe el reporte final y ARNI registra la decisión siguiente: avanzar, ajustar, pausar o redirigir.

## 6. Control de calidad

Antes de entregar resultados al cliente, ARNI debe validar:

1. Que la información esté ordenada y sea entendible.
2. Que los supuestos estén claramente señalados.
3. Que no se prometan ventas, permisos, contratos ni resultados garantizados.
4. Que los riesgos estén documentados.
5. Que las recomendaciones sean accionables.
6. Que los temas legales, fiscales, regulatorios, migratorios o aduanales estén marcados para revisión especializada.
7. Que cada siguiente paso tenga responsable y plazo sugerido.
8. Que el reporte distinga entre hechos, señales, inferencias y supuestos.

---

## 7. Reglas operativas

1. No iniciar acercamientos externos sin validar el mensaje con el cliente.
2. No presentar contactos como oportunidades confirmadas.
3. No prometer acceso, ventas, permisos, inversión o contratos.
4. No emitir opiniones legales, fiscales, regulatorias, migratorias o aduanales.
5. No avanzar a Business Matchmaking si el cliente no tiene preparación mínima.
6. No recomendar entrada a mercado si las señales son débiles o contradictorias.
7. Registrar siempre fuente, fecha y responsable de cada hallazgo.
8. Separar información confirmada de supuestos.
9. Mantener trazabilidad de reuniones, respuestas y decisiones.
10. Escalar temas especializados con aliados o expertos externos.

## 9. Semáforo de decisión

| Color    | Significado                                                        | Recomendación                                                        |
| -------- | ------------------------------------------------------------------ | -------------------------------------------------------------------- |
| Verde    | Hay señales positivas, cliente preparado y actores relevantes.     | Avanzar a Business Matchmaking, Pilot Validation o agenda comercial. |

| Rojo     | Baja preparación, mercado débil o alto riesgo.                     | Pausar, replantear mercado o redefinir oferta.                       |

## 12. Cierre operativo

El proceso de Soft Landing debe cerrar con una recomendación clara, no con una promesa comercial.

La pregunta final que debe responder ARNI es:

**¿Qué debe hacer el cliente ahora para entrar al mercado con menor incertidumbre y mayor control?**
