# Calculadora de Calificación de Oportunidades

Esta herramienta permite evaluar oportunidades comerciales de ARNI Consulting usando un sistema simple de puntuación.

## Objetivo

Ayudar a ARNI a priorizar prospectos, reuniones y oportunidades comerciales de forma más objetiva.

## Criterios de evaluación

Cada oportunidad se evalúa con 9 criterios.

| Criterio | Pregunta clave | Puntaje |
|---|---|---|
| Necesidad clara | ¿La contraparte tiene un problema o interés concreto? | 0-2 |
| Decisor identificado | ¿Existe contacto con capacidad de decisión o influencia? | 0-2 |
| Capacidad probable de pago | ¿La empresa parece poder contratar servicios de ARNI? | 0-2 |
| Urgencia | ¿Existe una razón para actuar pronto? | 0-2 |
| Fit con ARNI | ¿ARNI puede aportar valor real? | 0-2 |
| Alcance posible | ¿Se puede definir un servicio claro? | 0-2 |
| Información disponible | ¿Existe información suficiente para avanzar? | 0-2 |
| Riesgo manejable | ¿El riesgo comercial u operativo es aceptable? | 0-2 |
| Siguiente paso | ¿Existe una acción concreta después del contacto? | 0-2 |

## Escala de evaluación

| Puntaje | Significado |
|---|---|
| 0 | No cumple |
| 1 | Cumple parcialmente |
| 2 | Cumple claramente |

## Resultado

| Puntaje total | Clasificación | Acción recomendada |
|---|---|---|
| 15 a 18 | Prioridad A | Seguimiento inmediato |
| 10 a 14 | Prioridad B | Solicitar más información o madurar |
| 0 a 9 | Prioridad C | Observar o descartar temporalmente |

## Formato de captura

| Campo | Valor |
|---|---|
| Empresa | [Nombre de empresa] |
| Contacto | [Nombre del contacto] |
| Sector | [Sector] |
| Servicio probable | [Servicio ARNI sugerido] |
| Puntaje total | [0-18] |
| Clasificación | [A/B/C] |
| Próximo paso | [Acción concreta] |
| Responsable | [Nombre] |

## Ejemplo de uso

| Criterio | Puntaje |
|---|---:|
| Necesidad clara | 2 |
| Decisor identificado | 1 |
| Capacidad probable de pago | 2 |
| Urgencia | 1 |
| Fit con ARNI | 2 |
| Alcance posible | 2 |
| Información disponible | 1 |
| Riesgo manejable | 2 |
| Siguiente paso | 2 |
| **Total** | **15** |

Resultado:

**Prioridad A** — Seguimiento inmediato.

## Reglas de uso

- No clasificar como A si no existe siguiente paso claro.
- No preparar propuesta completa si falta información crítica.
- No usar la puntuación como sustituto del criterio profesional.
- Revisar riesgos antes de avanzar.
- Registrar supuestos cuando no haya información confirmada.

## Principio de la herramienta

La calculadora no decide por ARNI. Ayuda a ordenar criterios, proteger tiempo comercial y priorizar oportunidades con mayor probabilidad de avance.
