# Herramienta de Ficha de Producto

Esta herramienta permite estructurar la información mínima de un producto para análisis comercial, market readiness, importer scouting, price benchmark y preparación de reuniones B2B.

## Objetivo

Ayudar a ARNI Consulting a documentar un producto de forma clara, comparable y útil para evaluar su potencial comercial en un mercado objetivo.

La ficha de producto no sustituye documentación técnica, regulatoria o certificaciones formales. Su función es ordenar la información comercial necesaria para análisis y acercamiento inicial.

## Cuándo usar esta herramienta

Debe usarse cuando una empresa quiere:

- Exportar un producto.
- Presentar un producto a compradores.
- Preparar importer scouting.
- Evaluar market readiness.
- Realizar price benchmark.
- Calcular landed cost.
- Participar en una misión comercial.
- Preparar materiales para reuniones B2B.

## 1. Información general del producto

| Campo | Descripción |
|---|---|
| Nombre del producto | Nombre comercial o técnico |
| Empresa | Empresa fabricante o comercializadora |
| País de origen | País donde se produce |
| Categoría | Categoría comercial del producto |
| Subcategoría | Segmento más específico |
| Descripción breve | Explicación clara del producto |
| Uso principal | Cómo se consume, usa o aplica |
| Mercado objetivo | País, región o canal objetivo |

## 2. Presentación y empaque

| Campo | Descripción |
|---|---|
| Presentaciones disponibles | Tamaños, formatos o variantes |
| Peso o volumen | Medida por unidad |
| Unidades por caja | Número de unidades por caja |
| Caja master | Información de empaque secundario |
| Vida de anaquel | Duración estimada del producto |
| Condiciones de almacenamiento | Temperatura, humedad u otras condiciones |
| Código de barras | Si está disponible |
| Fotografías | Imagen del producto, empaque y etiqueta |

## 3. Información comercial

| Campo | Descripción |
|---|---|
| Precio base | Precio de referencia en origen |
| Incoterm | EXW, FOB, CIF, DDP u otro |
| Moneda | Moneda del precio |
| MOQ | Volumen mínimo de pedido |
| Capacidad de suministro | Producción o disponibilidad estimada |
| Canales actuales | Canales donde ya se vende |
| Clientes actuales | Tipos de clientes o referencias |
| Experiencia exportadora | Países o mercados atendidos |

## 4. Diferenciadores

Documentar los elementos que pueden hacer competitivo al producto.

Ejemplos:

- Origen.
- Calidad.
- Ingredientes.
- Diseño.
- Certificaciones.
- Historia de marca.
- Sostenibilidad.
- Precio.
- Innovación.
- Conveniencia.
- Formato.
- Sabor.
- Funcionalidad.

## 5. Certificaciones y cumplimiento

| Campo | Descripción |
|---|---|
| Certificaciones | Orgánico, Kosher, Halal, HACCP, ISO u otras |
| Registro sanitario | Si aplica |
| Etiquetado | Idioma, país, requisitos conocidos |
| Ficha técnica | Disponible / No disponible |
| Hoja de seguridad | Disponible / No disponible |
| Análisis de laboratorio | Disponible / No disponible |
| Requisitos especiales | Información relevante por mercado |

## 6. Competidores y referencias

| Campo | Descripción |
|---|---|
| Competidor directo | Marca o producto similar |
| Competidor indirecto | Producto sustituto |
| Rango de precio observado | Precio de referencia del mercado |
| Canal de referencia | Retail, specialty, marketplace, HORECA u otro |
| Diferenciación frente a competidores | Ventaja o desventaja principal |

## 7. Evaluación inicial ARNI

| Criterio | Evaluación |
|---|---|
| Claridad del producto | Alta / Media / Baja |
| Potencial comercial | Alto / Medio / Bajo |
| Nivel de preparación | Listo / Requiere preparación / No recomendable |
| Riesgo de precio | Alto / Medio / Bajo |
| Riesgo regulatorio | Alto / Medio / Bajo |
| Canal sugerido | Importador / Distribuidor / Retail / HORECA / Otro |
| Próximo paso | Acción recomendada |

## 9. Riesgos frecuentes

ARNI debe revisar riesgos como:

- Producto difícil de explicar.
- Falta de ficha técnica.
- Falta de precio base.
- Presentación no comparable.
- Vida de anaquel insuficiente.
- Falta de certificaciones relevantes.
- Empaque no adaptado al mercado.
- Falta de materiales en inglés.
- Capacidad de suministro limitada.
- Expectativas de precio poco realistas.
- Falta de responsable comercial.

## 11. Principio de la herramienta

Una ficha de producto útil no debe ser solo descriptiva. Debe permitir evaluar si el producto puede compararse, presentarse y defenderse comercialmente en el mercado objetivo.
