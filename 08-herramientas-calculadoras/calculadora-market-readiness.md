# Calculadora de Market Readiness

Esta herramienta permite evaluar si una empresa, producto o servicio está preparado para entrar a un nuevo mercado.

## Objetivo

Ayudar a ARNI Consulting a identificar si un cliente tiene condiciones suficientes para avanzar hacia investigación de mercado, importer scouting, soft landing, piloto comercial o propuesta formal.

La calculadora no garantiza éxito comercial. Su función es ordenar criterios y detectar brechas antes de invertir más recursos.

## Cuándo usar esta herramienta

Debe usarse cuando una empresa quiere:

- Exportar.
- Entrar a un nuevo país.
- Buscar importadores o distribuidores.
- Validar precio viable.
- Preparar ficha de producto.
- Evaluar canales de entrada.
- Participar en una misión comercial.
- Medir si está lista para un piloto comercial.

## 1. Criterios de evaluación

Cada criterio se evalúa de 0 a 2 puntos.

| Criterio | Pregunta clave | Puntaje |
|---|---|---|
| Producto claro | ¿El producto o servicio se entiende fácilmente? | 0-2 |
| Propuesta de valor | ¿Tiene diferenciadores claros frente a competidores? | 0-2 |
| Documentación comercial | ¿Cuenta con ficha, presentación o materiales básicos? | 0-2 |
| Precio definido | ¿Tiene precio base, referencia o estructura comercial? | 0-2 |
| Capacidad de suministro | ¿Puede atender demanda inicial o piloto? | 0-2 |
| Adaptabilidad al mercado | ¿Puede ajustar empaque, presentación, idioma o canal? | 0-2 |
| Evidencia comercial | ¿Tiene ventas, clientes, casos o experiencia previa? | 0-2 |
| Cumplimiento básico | ¿Cuenta con requisitos, certificaciones o información regulatoria mínima? | 0-2 |
| Seguimiento comercial | ¿Tiene responsable y capacidad de responder a prospectos? | 0-2 |
| Objetivo de entrada | ¿Sabe qué quiere lograr en el mercado objetivo? | 0-2 |

## 2. Escala de puntuación

| Puntaje | Significado |
|---|---|
| 0 | No cumple |
| 1 | Cumple parcialmente |
| 2 | Cumple claramente |

## 3. Resultado total

El puntaje máximo es 20 puntos.

| Puntaje total | Clasificación | Acción recomendada |
|---|---|---|
| 17 a 20 | Listo para scouting | Avanzar a búsqueda de contrapartes o piloto |
| 11 a 16 | Requiere preparación | Corregir brechas antes de avanzar |
| 0 a 10 | No recomendable por ahora | Preparar fundamentos antes de invertir en mercado |

## 6. Interpretación

### Listo para scouting

La empresa tiene condiciones suficientes para avanzar hacia búsqueda de compradores, importadores, distribuidores o aliados.

Acciones sugeridas:

- Preparar lista de prospectos.
- Validar canales.
- Ajustar mensaje comercial.
- Iniciar importer scouting.
- Preparar piloto comercial.

### Requiere preparación

La empresa tiene potencial, pero necesita fortalecer elementos antes de exponerse al mercado.

Acciones sugeridas:

- Mejorar ficha de producto.
- Ajustar propuesta de valor.
- Definir precio viable.
- Revisar empaque o presentación.
- Preparar materiales en inglés.
- Confirmar requisitos básicos.

### No recomendable por ahora

La empresa todavía no tiene condiciones mínimas para avanzar sin alto riesgo comercial.

Acciones sugeridas:

- Replantear producto o mercado objetivo.
- Construir materiales básicos.
- Definir precios.
- Validar capacidad interna.
- Preparar documentación mínima.

## 7. Brechas frecuentes

ARNI debe revisar especialmente estas brechas:

- Producto difícil de explicar.
- Precio no comparable.
- Falta de ficha técnica.
- Falta de materiales en inglés.
- Empaque no adaptado.
- Capacidad limitada de suministro.
- Falta de certificaciones o información regulatoria.
- Falta de responsable comercial.
- Expectativas poco realistas.
- Mercado objetivo demasiado amplio.

## 8. Uso futuro

Esta calculadora puede convertirse después en:

- Excel.
- Airtable.
- Formulario de diagnóstico.
- Dashboard de readiness.
- Calculadora web.
- Agente IA de diagnóstico comercial.

## 9. Principio de la herramienta

Market readiness no significa estar listo para vender. Significa tener condiciones mínimas para explorar el mercado con menor riesgo y mayor claridad.
