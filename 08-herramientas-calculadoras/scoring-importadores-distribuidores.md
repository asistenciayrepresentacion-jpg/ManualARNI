# Scoring de Importadores y Distribuidores

Esta herramienta permite evaluar y priorizar importadores, distribuidores, compradores o aliados comerciales potenciales.

## Objetivo

Ayudar a ARNI Consulting a identificar qué contrapartes tienen mayor compatibilidad con el producto, mercado objetivo y estrategia comercial del cliente.

El scoring no garantiza interés ni cierre comercial. Su función es priorizar contactos para usar mejor el tiempo de búsqueda, acercamiento y seguimiento.

## Cuándo usar esta herramienta

Debe usarse cuando ARNI esté realizando:

- Importer scouting.
- Distributor scouting.
- Buyer discovery.
- Preparación de agenda B2B.
- Priorización de prospectos comerciales.
- Validación de canales.
- Seguimiento posterior a misiones comerciales.
- Evaluación de aliados en mercado destino.

## 1. Información mínima de la contraparte

Antes de calificar, ARNI debe recopilar:

- Nombre de la empresa.
- País y ciudad.
- Sitio web.
- Tipo de contraparte.
- Sector.
- Canales atendidos.
- Productos relacionados.
- Marcas representadas.
- Cobertura geográfica.
- Contacto identificado.
- Evidencia de actividad comercial.
- Observaciones relevantes.

## 2. Criterios de evaluación

Cada contraparte se evalúa de 0 a 2 puntos.

| Criterio | Pregunta clave | Puntaje |
|---|---|---|
| Fit de producto | ¿Maneja productos similares o complementarios? | 0-2 |
| Fit de canal | ¿Atiende el canal objetivo del cliente? | 0-2 |
| Cobertura geográfica | ¿Opera en la región o mercado prioritario? | 0-2 |
| Experiencia con importados | ¿Trabaja con productos o proveedores extranjeros? | 0-2 |
| Portafolio compatible | ¿Su portafolio permite integrar el producto? | 0-2 |
| Señales de actividad | ¿Tiene sitio, redes, catálogo o presencia comercial activa? | 0-2 |
| Contacto identificable | ¿Existe contacto, email, formulario o responsable claro? | 0-2 |
| Tamaño o capacidad aparente | ¿Parece tener capacidad operativa suficiente? | 0-2 |
| Reputación o credibilidad | ¿Se percibe como empresa seria y verificable? | 0-2 |
| Potencial de seguimiento | ¿Existe una acción concreta para contactarlo o validarlo? | 0-2 |

## 3. Escala de puntuación

| Puntaje | Significado |
|---|---|
| 0 | No cumple o no hay evidencia |
| 1 | Cumple parcialmente o requiere validación |
| 2 | Cumple claramente |

## 4. Resultado total

El puntaje máximo es 20 puntos.

| Puntaje total | Clasificación | Acción recomendada |
|---|---|---|
| 17 a 20 | Prioridad A | Contactar primero |
| 11 a 16 | Prioridad B | Validar antes de contactar o contactar en segunda ronda |
| 0 a 10 | Prioridad C | Mantener como referencia o descartar temporalmente |

## 5. Formato de captura

| Campo | Valor |
|---|---|
| Empresa | [Nombre de empresa] |
| País / ciudad | [Ubicación] |
| Tipo de contraparte | [Importador / Distribuidor / Retail / Broker / Otro] |
| Sitio web | [URL] |
| Producto relacionado | [Categoría o producto] |
| Canal | [Canal atendido] |
| Puntaje total | [0-20] |
| Clasificación | [A/B/C] |
| Justificación de fit | [Razón de compatibilidad] |
| Riesgos o dudas | [Observaciones] |
| Próximo paso | [Acción concreta] |
| Responsable | [Nombre] |

## 6. Interpretación

### Prioridad A

Contraparte con alta compatibilidad y señales claras de actividad comercial.

Acciones sugeridas:

- Contactar en primera ronda.
- Preparar mensaje personalizado.
- Verificar contacto adecuado.
- Registrar seguimiento.
- Evaluar respuesta para posible reunión.

### Prioridad B

Contraparte con potencial, pero requiere más validación.

Acciones sugeridas:

- Revisar portafolio.
- Buscar contacto más adecuado.
- Validar canal.
- Confirmar si trabaja con productos importados.
- Contactar después de los prospectos A.

### Prioridad C

Contraparte con baja compatibilidad o información insuficiente.

Acciones sugeridas:

- Mantener como referencia.
- No priorizar contacto inicial.
- Descartar si no hay evidencia de fit.
- Revisar solo si faltan prospectos mejores.

## 7. Riesgos a considerar

ARNI debe revisar riesgos como:

- Sitio web desactualizado.
- Empresa sin contacto claro.
- Portafolio poco relacionado.
- Canal equivocado.
- Empresa demasiado grande para proveedores pequeños.
- Empresa demasiado pequeña para volumen requerido.
- Falta de señales de importación.
- Cobertura geográfica no alineada.
- Falta de evidencia comercial reciente.
- Confusión entre retailer, broker, mayorista e importador.

## 8. Relación con importer scouting

Esta herramienta debe utilizarse junto con el playbook de importer scouting.

El proceso recomendado es:

1. Identificar prospectos.
2. Registrar información mínima.
3. Calificar con scoring.
4. Clasificar A/B/C.
5. Preparar mensaje de acercamiento.
6. Dar seguimiento.
7. Registrar respuesta.
8. Convertir en oportunidad si existe interés real.

## 9. Uso futuro

Esta herramienta puede convertirse después en:

- Excel de scoring.
- Airtable de prospectos.
- Dashboard de importadores.
- Base de compradores por mercado.
- Agente IA de clasificación de contrapartes.
- Herramienta de preparación de agendas B2B.

## 10. Principio de la herramienta

Un buen scouting no depende solo de encontrar muchos contactos. Depende de identificar contrapartes compatibles, priorizarlas correctamente y preparar un acercamiento comercial con mayor probabilidad de respuesta.
