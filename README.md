# Manual ARNI Consulting

Repositorio operativo de ARNI Consulting para documentar servicios, procesos, playbooks, plantillas, prompts y herramientas internas.

## Objetivo

Centralizar el conocimiento operativo de ARNI Consulting para facilitar la preparación de propuestas, capacitación interna, diseño de servicios, uso de agentes de IA y seguimiento de proyectos.
