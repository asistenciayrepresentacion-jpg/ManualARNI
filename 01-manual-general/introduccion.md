# Introducción al Manual Operativo ARNI

Este manual tiene como objetivo documentar la forma de trabajo de ARNI Consulting.

## Propósito

Servir como referencia interna para organizar procesos, servicios, metodologías, criterios de análisis y herramientas de apoyo.

## Uso del manual

Este manual debe utilizarse para:

- Estandarizar servicios.
- Preparar propuestas.
- Capacitar colaboradores.
- Documentar procesos.
- Alimentar agentes de IA.
- Mantener trazabilidad de cambios.

## Principio operativo

ARNI Consulting trabaja como una consultoría boutique enfocada en inteligencia comercial, desarrollo de oportunidades, soft landing, análisis de mercado y preparación estratégica de empresas para expansión internacional.
